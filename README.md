### ¿Cómo instalar?

* Descargar el proyecto dentro de un directorio especifico

* Entrar al directorio del proyecto desde Consola/Terminal

    - Instalar dependencias: `mvn clean`
    - Instalar empaquetar: `mvn install`
    - Desplegar
